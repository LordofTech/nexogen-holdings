# Nexogen Group Limited - UI/UX Architecture

## 1. Concept: "The Synthetic Intelligence Framework"
A design language that balances high-utility terminal aesthetics with premium consumer luxury.

## 2. Visual Hierarchy & Spacing
- **Grid:** 8px soft grid system.
- **Margins:** 24px (Mobile), 80px (Desktop).
- **Typography:** 
  - Sora (Variable) for display: 48px/56px (H1), 32px/40px (H2).
  - JetBrains Mono for data: 12px/16px (Labels).
- **Interactions:** Subtle blur transitions, hover-activated glows, and scroll-triggered parallax sections.

## 3. User Journey (Mobile)
1. **Entry:** High-impact hero with WebGL liquid background and 3D wireframe interaction.
2. **Value Prop:** Horizontal scroll cards detailing "Infrastructure," "Intelligence," and "Deployment."
3. **Evidence:** A scrolling ticker of "System Logs" (live project updates).
4. **Action:** A persistent bottom-bar "Contact" trigger.
